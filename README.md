# NIFTY PULSE AUTO POSTING BOT

Telegram auto posting bot for NIFTY PULSE.

## Features

✅ Weekly scheduled posts  
✅ 5 daily auto posts  
✅ Photo + Text + URL support  
✅ Daily result image generator  
✅ Preview post system  
✅ Admin-only access  
✅ India timezone support

