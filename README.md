# 🤖 Telegram Auto Posting Bot

A fully automated Telegram posting bot that runs 24/7 using **Render Web Service**.  
This bot supports:

- Posting text, photos, and links
- Time conversion (24h → 12h AM/PM)
- Admin-only access
- Works continuously with no downtime

---

## 🚀 Features

- 🕒 Converts railway/24h time → normal AM/PM  
  Example: `18:30 → 06:30 PM`
- 🛡 Admin protected (only approved admins can use commands)
- 🔄 Ready for 24/7 hosting on Render
- 📡 Supports /start, /show_times and posting operations
- 📷 Easy future expansion for scheduled auto posting
